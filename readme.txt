A lightweight plugin that adds maintenance mode or a custom front page without modifying your WordPress theme.

== Description ==

Front Page Plug allows site owners and developers to:

* Enable maintenance mode while retaining admin access
* Replace the front page without editing the active theme
* Use a built-in template or an existing WordPress page
* Preview changes safely as an administrator
* Keep development work hidden while informing visitors

This plugin is intentionally lightweight, transparent, and privacy-respecting.
It does not track users, collect data, or include advertisements.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate **Front Page Plug** from the Plugins menu
3. Go to **Settings â†’ Front Page Plug**
4. Choose your desired mode and configuration

== Frequently Asked Questions ==

= Does this plugin modify my theme? =
No. Front Page Plug does not alter theme files in any way.

= Can administrators still access the site? =
Yes. Administrators always retain full access.

= Can I use an existing WordPress page as the front page? =
Yes. You can select an existing page from the plugin settings.

= Does this plugin track users or collect data? =
No. Front Page Plug does not collect analytics or telemetry.
